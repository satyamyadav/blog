### Blog  

*A hexo based blog setup deployed at http://satyamyadav.github.io/*

```bash
git clone https://github.com/satyamyadav/blog.git
git submodule update --init
npm install
```



install hexo:

`npm install hexo -g`

Build: 

`hexo generate`

Watch: 

`hexo generate -w`

Serve:

`hexo serve`

deploy:

`hexo deploy`
